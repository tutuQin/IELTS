# IELTS 四科总览

| 科目        | 时间      | 题量      |
| --------- | ------- | ------- |
| Listening | 30分钟    | 40题     |
| Reading   | 60分钟    | 40题     |
| Writing   | 60分钟    | 2篇作文    |
| Speaking  | 11-14分钟 | 3 Parts |

总考试时间：

约2小时45分钟

对于你目前的目标（海外技术工作、Web3、远程岗位），四科重要性可以理解为：

**Listening ≈ Speaking > Reading > Writing**

因为未来真正面试和工作时，最常用的是：

* 听懂会议（Listening）
* 开口沟通（Speaking）
* 阅读文档（Reading）
* 写邮件和文档（Writing）

但考试提分角度通常是：

**Listening + Reading 先冲 7.5 → Speaking 6.5+ → Writing 6.0~6.5**

这样最容易把总分拉到 7.0。
