- 时态注意：描述已故名人均使用过去时态

#### 二、知识小结

|              |                                                                                                            |                                                     |      |
| ------------ | ---------------------------------------------------------------------------------------------------------- | --------------------------------------------------- | ---- |
| 知识点          | 核心内容                                                                                                       | 考试重点/易混淆点                                           | 难度系数 |
| 描述人物的逻辑框架    | 分为外貌、性格、关系、影响四部分展开描述，适用于口语Part 2                                                                           | 从大到小的开头逻辑（如“There are many...but my father is one”） | ⭐⭐   |
| 小人物（普通人）描述范例 | 以父亲为例，结合外貌（tall and buff）、性格（warm-hearted）、共同活动（play jazz）、影响（纠正性格缺陷）                                      | 形容词需用动词支持（如“warm-hearted”对应“tells me to be kind”）   | ⭐⭐⭐  |
| 大人物（名人）描述范例  | 以Michael Jackson为例，涵盖背景（King of Pop）、外貌（good-looking）、成就（albums畅销）、社会贡献（帮助非洲儿童）                            | 时态一致性（已故名人需全程用过去时）                                  | ⭐⭐⭐⭐ |
| 话题卡灵活套用      | 小人物素材可覆盖：a good parent/a teenager you know/a talkative person等；大人物素材可覆盖：a famous athlete/a popular person等 | 同一素材适配多话题的关键：调整细节（如将“父亲”替换为“朋友”）                    | ⭐⭐   |
