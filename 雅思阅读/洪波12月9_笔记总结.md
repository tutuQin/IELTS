这是一份关于雅思阅读（IELTS Reading）备考策略的深度教学笔记，主要归纳了“真经派”阅读法在应对不同题型时的解题逻辑、定位技巧及同义替换核心规律。以下是课程内容的结构化总结：

### 一、 核心解题方法论

- **同义替换（核心考点）：** 雅思阅读考试的本质是对词汇改写能力的考察。解题关键不在于词汇量大小，而在于建立题干与原文之间的语义关联。
    
- **“三行定位法”与排除法：** 当无法直接定位时，聚焦关键词所在行及其上下三行，通过划掉已知词汇，锁定不认识的单词作为潜在答案。
    
- **真经派阅读法：** 强调高效阅读与解题同步，建议先做细节题，通过细节积累的关键词和语义线索反推中心思想题（如Heading题），避免为单题重读全文。
    

### 二、 主要题型应对策略

|**题型**|**解题重点**|**关键技巧**|
|---|---|---|
|**Summary填空**|空格前后的名词/动词同义替换|聚焦上下文逻辑，关注词性匹配（复数/单数）。|
|**多选题**|定位原文的并列结构|寻找选项关键词与文中并列要素的改写对应，优先选具体选项。|
|**T/F/NG判断**|区分排他性矛盾、未提及与同义改写|TRUE题多为单句改写；FALSE题为排他性矛盾；NG题为原文缺失或无法验证。|
|**单选题/主旨题**|驳斥关系、实验结论、作者意图|留意转折词（while/but/still），区分细节信息与段落中心论点。|

### 三、 重点命题规律与陷阱

- **命题递进逻辑：** 题目设计遵循“单句改写 → 相邻句子归纳 → 不相邻内容总结”的难度梯度。
    
- **职业后缀与学术术语：** 识别职业后缀（如-ist）和术语创造标志（如coined the term），是快速定位答案的捷径。
    
- **真题严谨性：** 指出剑桥真题偶有设计瑕疵（如答案多元化或题目错误），强调考场策略是“选择最符合逻辑和最具体的选项”，不必过分纠结。
    
- **空域与空间逻辑：** 在处理复杂逻辑类文章（如空域控制、生态依赖）时，需在脑中构建空间或关系模型，而非仅进行字面翻译。
    

### 四、 备考建议

- **批判性思维：** 反对机械背字典式的记忆法，强调在语境中学习词汇。
    
- **考点词清单：** 利用《考点词真经》系统复习剑桥4-10中的同义替换对，这是掌握命题规律的核心工具。
    
- **策略灵活性：** 灵活处理NB规则（提示重复使用），利用文章结构和题量饱和度（排除法）来提高正确率。
    

**总结：** 本课程笔记强调**从“词汇对齐”到“逻辑理解”的思维转变**。通过对雅思阅读中高频出现的逻辑结构（并列、因果、转折）和改写范式的精准拆解，帮助考生在有限时间内通过精准定位和逻辑推导，跨越语言障碍，提高得分效率。

这份课程笔记本质上是一套**雅思阅读真经派阅读方法课**，核心不是讲文章内容，而是在讲**雅思阅读各题型的解题逻辑、定位技巧、同义替换规律以及考试策略**。

# 一、课程核心思想

**定位 + 改写**

1. 找到题目对应原文位置（定位）
    
2. 找到题目与原文之间的同义替换（改写）

# 二、最重要的知识点：同义替换

这是整节课的核心。

例如：

|题目|原文|
|---|---|
|modern technologies|new technology|
|domestic|home|
|become aware|realize|
|importance|essential role|
|hard work|effort|
|accuracy|correct answers|
|finding|detection|
|refer to|talk about|

老师强调：

### TRUE题

几乎都是：

> 原文 + 3组以上同义替换

例如：

原文：

> We begin to realize its essential role.

题目：

> People become aware of its importance.

对应：

- begin → become
    
- realize → aware
    
- essential → important
    

所以选 TRUE。

# 三、Summary填空题解题法（重点）

课程认为：

> Summary本质就是填空题。

## 核心原则

不要看空格。

重点看：

- 空格前
    
- 空格后
    

的重要名词和动词。

例如：

```
pain ____ on a leg
```

不要研究空格。

而是找：

- pain
    
- leg
    

然后去原文寻找同义替换。

---

## 解题步骤

第一步：

定位关键词

第二步：

找同义替换

第三步：

锁定答案附近三行

第四步：

利用排除法

---

老师提出：

### 三行原则

答案通常出现在：

> 关键词上下三行以内

---

# 四、多选题解题法

课程强调：

## 找并列结构

例如：

```
A and B
A, B, C, D
```

或者：

```
coupled with
```

```
together with
```

```
as well as
```

这些地方最容易出多选题。

---

例如：

### 日本数学成功原因

原文：

- 重视教育
    
- 数学是必修课
    
- 强调努力
    
- 强调准确率
    

四个并列原因。

答案一定从这里选。

# 五、TRUE / FALSE / NOT GIVEN 系统方法

这是课程讲得最多的部分。

---

## TRUE

特点：

原文有

并且进行了同义替换。

例如：

```
modern technology
```

改成：

```
new technology
```

TRUE

---

## FALSE

特点：

原文有

但是意思相反。

例如：

原文：

```
最喜欢咖啡
```

题目：

```
最喜欢茶
```

FALSE

---

## NOT GIVEN

特点：

题目增加了原文没有的信息。

例如：

原文：

```
私立学校现代化
```

题目：

```
私立学校比公立学校更现代化
```

原文没比较。

NOT GIVEN

---

# 七、Heading题（段落标题题）

课程强调：

### 不要先做Heading

正确顺序：

```
先做细节题
↓
再做Heading
```

因为：

细节题做完后，

已经知道段落在讲什么。

Heading就变简单了。

---

## Heading选择原则

必须覆盖：

### 段落核心关键词

例如：

盲人 + 战争

正确标题必须同时体现：

- Blind people
    
- War
    

缺一不可。

# 九、逻辑猜题法（应急技巧）

时间不够时：

### 优先选长段落

因为：

长段落：

- 信息量大
    
- 出题点多
    

所以更容易对应答案。

---

### 排除已出题饱和段落

例如：

D段

E段

已经出了很多题。

新题大概率不会继续出。
